## Как посмотреть?

У вас должны быть установлены:

- [ruby](https://www.ruby-lang.org/ru/downloads/)
- [nodejs и npm](https://nodejs.org/)
- [bower](http://bower.io/)

Далее выполняем:

```
sudo npm i
sudo bower i
```
После выкачивания всех зависимостей проекта запустите задачу `index.html`.
